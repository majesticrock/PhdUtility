Detailed Description

Represents a collection of momentum symbols with associated operations.

This class represents momenta. It includes addition and substraction operators as well as a bool add_Q. is defined as , i.e., for all even . Besides the normal operators in which you specify the class members, you can also pass a string like "3k+l-p" to the constructor to create that specific momentum. If you want to add here, you can do so by passing true to the same constructor as a second argument.
