Detailed Description

An abstract base class for Wick symmetries.

There may be some symmetries that simplify your results, e.g., . These symmetries can be implemented by inheriting from the WickSymmetry class and defining the member function virtual void apply_to(WickTerm& term) const. Then create a std::vector<std::unique_ptr<WickSymmetry>> symmetries and make use of polymorphism by calling clean_wicks(wicks,symmetries). There are the following predefined symmetry operations:

SpinSymmetry
Changes all spins of the operators in term to .

InversionSymmetry
Flips the momenta in such a way, that the first momentum in a term is always positive, i.e., is changed to while would stay unmodified.

PhaseSymmetry
Takes a list of OperatorType as template arguments. Removes any dagger from all operators with a type from the list. Example: PhaseSymmetry<SC_Type,CDW_Type removes the dagger from SC_Type and CDW_Type operators.
